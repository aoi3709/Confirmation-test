# Draw.io Mermaid Plugin

This draw.io plugins integrates the mermaid diagram generator.

Mermaid supports:
* Pie charts
* Sequence diagrams
* Gantt diagrams
* State diagrams
* Flow charts
* Class diagrams

via a simple markup language.
